Timechicken项目

一、项目要约
1.项目严格遵循阿里规范
2.项目提交备注需完整填写  如：* 提交内容
3.项目方法需要遵循restful 风格。项目方面需要详细的提供方法说明
4.尽量减少冗余代码， 若有冗余代码，请抽取成共用方法或代码块
5.代码必须有注释（非常重要）

二、数据库创建规范
1.除非业务需要，或出于性能优化考虑，尽量少的使用冗余字段。
2.数据库字段命名规范 使用小写字母与下划线 如：user_id
3.数据库表与字段必须添加备注
4.数据库创建时，需要考虑索引以提升效率。
5.枚举类型字段，需有备注说明枚举值所代表意义 如 state: 1正常 2停用 3删除

三、项目基本技术架构
1.项目使用Maven做Jar包管理;
2.项目使用SSM 框架；(SpringMVC Spring Mybatis)
3.项目使用ES 搜索引擎 (ElasticSearch)
4.项目使用Mybatis Generator 自动生成。
5.项目使用Redis 缓存。
